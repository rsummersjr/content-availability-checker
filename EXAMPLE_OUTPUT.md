## What the Publication Date Represents

| Platform | Date Meaning |
|----------|--------------|
| **YouTube** | Date when the video was uploaded to YouTube |
| **YouTube Music** | Same as YouTube (video upload date) |
| **Deezer** | Album or track release date |
| **Apple Music** | Official track release date |
| **Amazon Music** | N/A - No API |
| **Tidal** | N/A - No API |

## Date Format
- Dates are displayed in a human-friendly format: **"Month DD, YYYY"**
- Examples: "January 15, 2024", "November 21, 1975"
- If a date is not available from the API, the field is simply not displayed
